关于本站

基于 Django 构建的个人博客。文章以 Markdown 文件存储，无需数据库，一键构建静态站点部署到 GitHub Pages。

功能

  • Markdown 渲染：LaTeX 数学公式、JetBrains Mono 代码高亮、删除线、表格
  • 8 种主题色：青绿 / 蓝 / 玫红 / 橘橙 / 紫 / 草绿 / 灰蓝 / 天蓝，一键切换
  • 智能搜索：多关键词加权打分，文章和项目分 tab 展示
  • 标签归档：点击标签胶囊跳转归档页，标签云浏览
  • 侧边目录：自动生成，滚动高亮
  • 文章 & 项目主页：独立列表页，带翻页导航

技术栈

  • 后端：Django 6.0
  • 内容：Markdown + YAML front matter
  • 前端：纯 CSS（HarmonyOS Sans SC 字体），无 JS 框架
  • 部署：PythonAnywhere / GitHub Pages
