Day 1 A. 幸运数字

直接贪心。判断一个数 \(x\) 是不是幸运数字，只需要把能取到 \(x\) 的全取到 \(x\)，不能取到 \(x\) 的就随他去，最后判断中位数是否是 \(x\) 即可。通过离散化可以知道本质不同的 \(x\) 只有 \(O(n)\)。

上面的判断可以用双指针来实现。总时间复杂度 \(O(n\log n)\)。

Day 1 B. 追忆

这道题花了我一节自习课，最后还是在回家的路上想出来的。

实际上代码难度不大，想清楚的话写的很快而且代码也很短。

看到有向图可达性，实际上能够解决这个问题的基本上就只有 bitset。

由于我刚刚拿到题面的时候还看不到特殊性质，所以我一开始想的时候就自己捏了个没有修改的特殊性质（实际上就是 AB）。当时的想法就是 bitset 存放每个点能够到达的所有点的 b 值，然后用莫队找到对应 a 范围内的点，再 bitset 查一次得到最大的合法的 b。

但是这个想法并不能很好的迁移到 b 会动态变化的原问题中。

然后我又对着这个东西想了很久，一直去想对于一个点怎么维护它所有能够到达的合法点集。这样的做法实际上是行不通的。

但当我们换一个维度去想问题——我们是否可以找到每个点能够贡献到的所有询问呢？这道题其实就 迎刃而解 水到渠成了。

考虑根号重构。把询问分成 \(\sqrt n\) 个块，每个块统一处理。对于每个块内被修改过 \(a_i/b_i\) 的点，我们可以直接暴力处理，因为它们总共就 \(O(\sqrt n)\) 个。

对于其它点，它们整个块内都没被修改过，相当于静态询问。我们考虑对于每个点怎么用 bitset 维护它所有能够贡献到的合法询问。首先是可达性的限制，跑一遍拓扑排序即可。这一部分是 \(O(\dfrac{n\sqrt n}{\omega})\) 的。

然后是 a 区间的限制。有两种做法，一种是注意到本质不同的 a 只有根号种，对于根号种值域每个开一个 bitset 预处理它能够贡献到的询问，然后对每个点就 and 一下对应值域的 bitset 就行。另一种做法就是直接离线按照 a 排序然后差分处理。这一部分都是 \(O(\dfrac{n\sqrt n}{\omega})\) 的。

最后，我们对于所有点按照 b 从大往小排序，每次找到没有被标记过的而且它能够贡献到的询问，对其进行标记并更新答案即可。

总时间复杂度 \(O(\dfrac{n^2}{\omega}+n\sqrt n)\)。

实际上我只花了 20min 就把代码写完了，但是卡常卡了一个半小时。最后解决常数问题的办法出乎意料的简单——实际上我之前的代码对于被修改过的点和没有被修改过的点是开了两组 bitset 分开处理的。后来发现这两组 bitset 需要实现的功能是本质相同的，于是只需要开一组 bitset 就行了。常数马上就减了一大半。

Day 2 A. 推箱子

不知道为啥现在还要考这种大模拟的题。

直接贪心。按照时间顺序先后处理每个箱子。处理每个箱子时，我们需要把它连带着和它相邻的一连串箱子全都搬到某块地方去。

类似于珂朵莉树，用 set 存储三元组 \((l,r,p)\) 表示 \(l\sim r\) 号箱子目前在 \(p\sim p+r-l\) 这块地方。再存一个全局时间戳 \(tim\)。每次搬箱子的时候，我们都需要合并一段连续的三元组。所以总的时间复杂度是 \(O(n\log n)\) 的。

代码真的很好写，花了 30min 不到。但是省选考场遇到这种题目确定心态比较容易崩，而且做法也不太能想清楚。能够冷静且淡定地写完整道题目的真的需要特别强的心理素质。

Part I. 信息学大厦，正在崩塌！

实际上这是一个非常有意思的事情。起因是这样一道题目。

先把题目翻译成人话：（仅第三问）一张竞赛图如果没有入度为 0 的点，那么一定存在三元环。

当时我的想法就是它的逆否命题：如果一张竞赛图没有三元环，那么一定有入度为 0 的点。

容易拓展：一张竞赛图如果没有三元环，那么一定没有任意环。所以它是个 DAG。

那么，为啥一个 DAG 一定有入度为 0 的点呢？？？

你可以说：一个 DAG 可以拓扑排序，那么拓扑排序中编号为 1 的点肯定没有入度啊。（实际上如果你清楚的话这就是循环论证，肯定证不出来）

但是一个 DAG 为什么可以拓扑排序呢？？？完蛋了。

实际上，信息学中我们接触的绝大多数结论都是基于感性理解的。图论是其中一个非常大的重灾区。但是随着 OI 的不断发展，对各种算法的考察也更加深入，我们不仅需要一个算法是怎么用的，也需要知道它的原理是什么。

下面列举出了一些证明被「感性理解」一笔带过的东西，我们来一个一个地解决它们：

  1. 为什么有向无环图可以拓扑排序？

如果一个 DAG 存在入度为 0 的点，那么一定可以拓扑排序。根据拓扑排序的过程，我们考虑归纳。每次删掉任意一个入度为 0 的点和它的所有出边，那么剩下的还是 DAG。所以我们只需要证明：

  1. 为什么有向无环图一定有入度为 0 的点？

    我们只需要证明有向无环的竞赛图一定有入度为 0 的点即可。因为任意有向图都可以由竞赛图删边得到。

    如果一个为 DAG 的竞赛图没有入度为 0 的点，那么我们给每个点找一个前驱，显然每个点都能找到。我们从任意一个点出发，不断去跳它的前驱，根据抽屉原理，至多跳 \(n\) 次就可以跳到一个重复的点，也就存在了环。这就导出了矛盾。

    所以一个 DAG 一定存在入度为 0 的点。（感谢 @freeplume 提供的思路）

    不过上面的证明还埋下了一个小小的雷：我们都知道，每个点入度为 1 的有向图一定是一棵内向基环树。但是它为什么会长成这种结构呢？（请读者自证）

  2. 为什么竞赛图如果没有入度为 0 的点，一定存在三元环？

上面已经说过了，一个 DAG 一定存在一个入度为 0 的点。

所以竞赛图没有入度为 0 的点，它就一定存在环。竞赛图如果存在环那肯定存在三元环（请读者自证）。

  1. 为什么所有点度数为偶数的连通图，一定有欧拉回路？

问题转化为能否用若干个简单环覆盖所有的边。

先证：对于一个图，如果每个点有邻边，那么一定存在一个简单环。

给每个点指定一个出点，把它到出点的边变成有向边，那么一定会张成一棵内向基环树，也就一定存在环。

如果还存在边，我们就按照这个方式找到环，然后把它删去，每个点的度数依然是偶数，递归即可。如果两个环存在公共点，我们就可以把这两个环拼在一起，不断拼接，这样我们就构造出了一条欧拉回路。

  1. 为什么二分图没有增广路就是最大匹配？

必要性很简单，如果存在增广路，就将路径上的所有边的匹配状态反转过来，这也是匈牙利算法的原理。

然后是充分性：还是一样，考虑逆否命题：没有达到最大匹配的二分图一定存在合法增广路。

考虑一个不是最大的二分图 \(G\) 和一个匹配数比它大 1 的二分图 \(G'\)。对于一条边，如果它在 \(G\) 中是非匹配边，\(G'\) 中是匹配边，我们就从右往左定向。如果它在 \(G\) 中是匹配边，\(G'\) 中是非匹配边，我们就从左往右定向。由于二分图和匹配的特殊性（可以用排列来类比），这张图一定会形成若干个简单环，若干条起终点在同一侧的简单链，若干条从左往右的简单链和若干条从右往左的简单链。

由于新图比原图的匹配数大 1，那么从右往左的链个数至少比从左往右的链个数多一条（也就是至少有一条）。我们随便取一条链，那么它的起点和终点必然是非匹配点，同时一定形成一条增广路，证毕。

  1. 为什么最大匹配等于最小点覆盖？

最大流最小割定理：最大匹配对于最大流，最小点覆盖对应最小割。

  1. 为什么 \(\gcd(a,b)=\gcd(b,a\bmod b)\)？

先证：\(\gcd(a,b)|b\)，而且 \(\gcd(a,b)|(a\bmod b)\)。 我们设 \(a\bmod b=r\)，则有 \(a=kb+r\)（ \(k\) 为非负整数 ）。同时设 \(g\) 为 \(a,b\) 的最大公因数。 \(\because g\mid a,g\mid b,\ \ \ \therefore g\mid kb+r,\ \ \ \therefore g\mid r\) 所以，\(g\) 既是 \(a,b\) 的公因数，也是 \(b,r\) 的公因数。 再证：不存在任意一个大于 \(\gcd(a,b)\) 的数 \(c\) 满足 \(c\mid b\) 且 \(c\mid (a\bmod b)\)。 使用反证法：如果存在 \(c>\gcd(a,b)\) 且满足 \(c\mid b\) 且 \(c\mid r\)。 \(\because a=kb+r\) 且 $c\mid b, c\mid r, \therefore c\mid a, $ 又 \(\because c\mid b,\ \ \ \therefore c\) 是 \(a,b\) 的公因数。 但又因为 \(c>\gcd(a,b)\) ，矛盾，所以不存在任何大于 \(\gcd(a,b)\) 的数满足条件。

综上，\(\gcd(a,b)=\gcd(b,a\bmod b)\)。

当然有一个更简单的方法是证明 \(\gcd(a,b)=\gcd(a,b-a)\)，虽然我写上面的东西的时候没想到

  1. 为什么无向图的 dfs 树只有树边和返祖边？

考虑按照后序遍历的方式进行归纳。叶子结点不可能再连出边。如果归纳到一个点，说明它子树中所有点都已经遍历完。如果还存在没有遍历到的边的话，首先它不可能通向我后面才遍历到的点。其次它也不可能遍历到处理祖先之外其它在它之前遍历的点，因为我们之前的点已经归纳完了。那它要么是通向子树的，要么是通向祖先的，肯定是一条返祖边。

  1. 为什么一张图是二分图和它没有奇环等价？

先考虑必要性：如果可以进行黑白染色，那么假设存在奇环，那么这个奇环中必然存在一条边连接的两个点颜色相同。

再考虑充分性：证明它的逆否命题：不是二分图必然存在奇环。不是二分图意味着对于任意一种黑白染色方案，都必然存在一条边连接的两点颜色相同。我们考虑这样一种染色方案：提取它的 dfs 树，将深度为奇数的点染黑，深度为偶数的点染白。那么由于它不是二分图，那么必然有一条边的两点颜色相同。这条边显然不能是树边，那么就只能是返祖边。那么这两个点的深度奇偶性相同，再加上这条边就会形成一个奇环。

机房是一个让人比较怀念的地方，虽然高一已经从我们原来的位置搬到了五楼，但是那种感觉还是在的。很有意思，我在机房的效率可能比在教室里面还高一些。

在此之后，每次月考前后，一些节假日的时候，有的时候数学老师晚上安排考试，或者单纯不想去教室自习，我都会去机房。我也不知道刘晖为啥要在晚自习安排那么多考试，有的考试感觉意义真不大，比如排列组合或者概率期望或者数列的考试，太没意思了。

最近一段时间驻扎在机房的时间比较多，一个是五一放假学校封校了我连我们班在干什么都完全不知道，另外一个是接下来要说的。

这段时间我感觉自己的学习已经逐渐到达了一个「过拟合化」的阶段：很多题用二级结论可以一眼秒掉，有一些大题可以直接默写，见到了大量的重复的且同质化的题目。

但是，面对一些基础题反而出现了不知所措的情况，有的时候读题还会主观臆断自己添加或者忽略一些信息。最严重的症状发生在英语：做题依赖于自己的感觉，有的时候会过度推断选项，很多明显错误的选项都会被自我催眠成正确的选项。

虽然从成绩上看不出来什么，但是这的确是一件比较可怕的事情。

所以，大概是几周之前，我就做了一个决定——限制自己的做题量。这应该是一个相当大胆的决定，我也没有把握它能不能够起效果。

在五一放假的这几天，我一次班级自习都没参加过。当然还发生了一些比较大的事情，这个目前不太能说。

刚好五一的时候雅礼在办湖南省队集训，我也去看了看。

哦，我还验了雅礼场的题目。题目质量都很好，就是放在一起有点恐怖。有一个小插曲：正式雅礼场的题目顺序是在前一天临时改动了的，因为我当时倒序开题，把原定的 T3 和 T2 都切掉了，但是 T1 确实是束手无策而且完全想歪了。所以正式考试的时候，T1 和 T2 被换了顺序，从考试结果来看这个决定应该是比较正确的。希望下次省队集训的时候，雅礼场是现役的雅礼同学出的题。我也没想到在省选之后还可以再接触 oi，现在看来 oi 其实还挺有意思的。

我也做了另外一个决定——我要开始跑步啦！！！！！

体育运动我从高中以来一直闭口不谈的东西。我们教练在我学 oi 的时候曾多次要求我做体育运动，很遗憾每次我都把它搪塞了过去。

我也不知道为啥我这么不喜欢体育运动。可能是因为我在学 oi 的时候身体一向比较差。我曾经创造过一个月感冒三次的纪录。每次 NOIP，省选或者是 NOI 之前我的身体都一定会出一些问题。大多数时候是感冒+发烧，偶尔是肚子痛。NOIP2023 的前一周，我还喜提智齿发炎 + 感冒 + 发烧 + 失眠的大礼包，虽然对我成绩好像没啥影响。

在高中阶段，我的体育成绩完全可以用依托答辩来形容。在高中为数不多的体测中，我每次跑 1000m 的时候，都是抱着「不上救护车就是胜利」的目标去跑的。跑完之后往往会有种天旋地转的感觉，缓一个上午才能够缓过来。

我曾经在上一篇 OI 后日谈提到过浅羽悠真的人物赏析。有的时候我会觉得：他体弱多病，我也体弱多病，我觉得我和他还是挺像的！实际上我和他还是有本质的区别：他是先天的，我很有可能是自己作的。

我也不知道我为啥会想着在高考前的 50 天开始跑步，但我就是开始跑了。生活，太奇妙了。

如果问我一个确切的原因，那就是有一天我从机房下晚自习的时候碰到了高二，不经意地（有可能是有目的地）问了问他们有没有跑步，得到了一个他们每天九点会到操场跑步的结论。

晚上九点，我就会从机房跑出去，拉上高二的同学（大多数时候也有初三的同学）一起跑步。

第一次跑步意料之外的顺利。在我问教练可不可以出去跑步的时候，她毫不犹豫地同意了，还问我为啥这个时候突然想起来要跑步了。生活，很神奇吧。操场上也有一些偷偷从晚自习溜出去的同学在进行各种包括跑步的运动。当然也有一些老师，但是并没有人管我们。

意料之外的顺利还包括跑步的过程。跑步实际上并不是一个很难的东西。如果能够迈出去第一步，那么肯定也能够迈出去第二步，第三步，第更多步。晚上的操场相当的安静，不用担心被一颗突然飞过来的足球撞到，也不用绕着正在训练的体育生。在你身边的是只有一些树，天上的月亮，还有一起跑步的同伴。

我从来没想过自己能够在完全自愿的情况下跑出一个比 1km 更长的距离。但是生活就是很奇妙。第一天我跑了四圈，虽然也只有 350×4=1400m，应该是一个相当短的距离，但是收获的满足感是相当多的。

在这之后的几天内，我每跑一次就会加上一圈，从五圈，六圈，再到七圈，八圈，再到 3km。虽然 3km 这个距离还是挺短的，但是我已经相当满足了。在第二次跑步的时候，我已经能够在配速和路程上都超过了高二的同学。看来高二信息组的身体也不是很好。

在某一个星期天的晚上，放学的时候我和 @Sentoayaka 一起回家。走到操场的时候他突然把书包放下来跑步，我表示我也想跑。我能看到那个时刻他绝对是相当的震惊的。一个一万年都不愿意跑步的人怎么今天突然就开始跑步了？（其实我也觉得挺奇怪的）

然后我们就开始了跑步。跑的时候，我还遇到了其它一起来跑步的同学，比如 @天命之路 和 @智子。我心想：终于到了一个展现自我的时刻！事实证明这个「展现自我」相当的成功：

在第三圈的时候，@Sentoayaka 停下来了。在第五圈的时候，@天命之路 也停下来了。而我，跑了八圈！感觉你们的身体也没比我好多少啊

事后我的同学们都对我表现出了惊讶且诧异的态度（换我我也会）。

当然最疯狂的一次还是在五一放假的时候。有一天晚上，我也不知道为什么，整整绕着操场跑了半个小时。可能那时的我大抵是疯掉了，但是事实就是如此。生活，很神奇吧。

写到这里突然想起来了我要跑步还有一个原因：初三的时候，我们的体育老师经常给我们布置绕着操场跑 8~9 圈的任务。很遗憾的是，每一次我要么是跑三圈就结束了，要么根本就没跑。我也不知道从来没有跑过完整的九圈是不是我心里的一个遗憾，但以前的我绝对想不到的是这个东西第一次完成发生在高考的前 40 天。

生活，很神奇吧。
