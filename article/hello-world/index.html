个人主页，堂堂回归！

旧版个人主页已经很久没有更新了。正好趁这个暑假，重新搭一个博客，把想写的东西都写下来。

这个博客

之前用 Hexo 搭过一个博客，写了一段时间就荒废了。这次用 Django 重新写了一个，自己掌控每一个细节。

功能方面：

  • 文章采用 Markdown 格式，支持 LaTeX 公式、代码高亮、删除线
  • 8 种主题色，点击切换
  • 标签归档 + 智能搜索
  • 项目页自动拉取 GitHub README
  • 侧边目录，滚动高亮
  • 一键构建静态站点，部署到 GitHub Pages

博客本身是开源的：Graphcity123/GraphcityBlog

我在做什么

  • 歌手信息检索系统（Singer2026）：清华大学程序设计训练实验一。爬了网易云 4718 首歌的数据，做成了一个完整的 Django Web 应用，带用户系统、收藏、搜索、数据分析。
  • 这个博客：也就是你正在看的东西。
  • 偶尔写写题解、笔记之类的，不定期更新。
